3. TikTok API Usage

Vanishing Roads uses the TikTok Content Posting API solely to upload videos to our own TikTok creator account. We request only the minimum permissions necessary for this purpose. Specifically:

– We use the video.publish scope to post videos to our own channel.
– We do not access any other user's account data, profile information, or content.
– We do not use TikTok Login Kit to authenticate third-party users.
– No TikTok user data is collected, stored, or shared.
